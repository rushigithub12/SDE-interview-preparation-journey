## 🎯 How to Use This Repository

### 1️⃣ **For Beginners**
Start with `JS_problems_1/` to build fundamental concepts:
```bash
cd JS_problems_1/
node isPrimeNumber.js
node palindrome.js
```

### 2️⃣ **For DSA Interview Prep**
Progress through `DSA/` folder systematically:
```bash
# Start with Arrays
cd DSA/Arrays/
node findMaxElement.js

# Move to Binary Search
cd ../BinarySearch/
node searchInRotatedSorted.js
```

### 3️⃣ **For LeetCode Practice**
Implement and test solutions:
```bash
cd leetcode-problems/
node twoSum.js
node rotateArraybyK.js
```

### 4️⃣ **For Company-Specific Prep**
- **Google, Amazon, Microsoft:** Focus on DSA + LeetCode-Medium/Hard
- **Startups:** GeeksForGeeks + Striver Problems
- **JavaScript Roles:** NamasteJS + React Machine Coding
